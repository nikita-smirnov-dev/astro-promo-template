# Юбилейное Промо-Приложение для Рок-группы "Silence Tends To Destroy"

Полноэкранное SPA (Single Page Application) промо-приложение, созданное специально к 10-летнему юбилею музыкального релиза рок-группы. Проект разработан как интерактивная цифровая карточка релиза, адаптированная под мобильные и десктопные устройства.

## Особенности проекта

- **Архитектура SPA на хэшах**: Быстрое переключение экранов (`#hero`, `#media`, `#details`) без перезагрузки страниц и лишней нагрузки на браузер.
- **Синхронизированный Музыкальный Плеер**: Полнофункциональный аудио-плеер с единым аудио-движком. Управление треками доступно как из основной медиа-панели, так и через компактный виджет, который автоматически появляется при уходе со страницы воспроизведения.
- **Строгая валидация данных (Zod)**: Вся информация о релизе, трек-листе и метаданных валидируется с помощью Zod-схем, что обеспечивает 100% безопасность типов (Type Safety) данных на этапе сборки.

## Стек технологий

- **Фреймворк**: [Astro]
- **Язык**: [TypeScript]
- **Стилизация**: Sass / SCSS
- **Валидация данных**: [Zod]
- **Иконки**: `astro-icon`

## Команды для запуска

Все команды выполняются из корня проекта через терминал:

Установка зависимостей:

```bash
npm install
```

Запуск development сервера:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Предпросмотр production build:

```bash
npm run preview
```
